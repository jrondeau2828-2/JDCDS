# Construire une partie
Aller sur https://tcg-arena.fr/build et aller directement à la partie 4, mettre le lien vers le game.json voulu.
! le lien doit être le lien raw du fichier sur github. Ex: https://raw.githubusercontent.com/jrondeau2828-2/JDCDS/refs/heads/main/v0.1/game.json.
Utiliser le lien fournis pour jouer.
